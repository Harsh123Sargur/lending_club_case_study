## General Information
- The intent of the project is to understand all the variable which will have significance impact on loan payment by a user. 
- This analysis would be done on the data set by a bank which has information about loan taken by different users and the loan status and other variables. 
- With This analysis we are coming to derive at the parameters on which the bank should be watchful before lending loan to any person.
